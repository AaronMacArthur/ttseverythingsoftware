# TikTok Webcast Proto Schemas

This folder is intentionally empty.

TTS Everything does not ship protobuf schemas copied from TikFinity, TikTok-Live-Connector, `@zerodytrash`, or related projects. The built-in TikTok Live source works in raw capture and generic protobuf inspection mode without schemas.

If the app developer later supplies legally obtained or self-authored `.proto` files, `WebcastSchemaDecoder` can load them from this folder or from the local diagnostics proto folder.

Do not add connector-derived schemas here.
